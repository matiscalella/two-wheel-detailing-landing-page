## About the Project

Two Wheels Detailing is a fictional responsive landing page developed as part of a university course (English II), focused on frontend development and technical presentation in English.

The objective was to build and present a complete landing page using modern frontend tools, including interactive behavior implemented with JavaScript.

## Features

- Fully responsive layout
- Light / Dark mode toggle
- Custom theming using CSS variables
- Pricing highlight animation
- FAQ accordion
- Structured and maintainable CSS architecture

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)

## Learning Outcomes

Through this project, I strengthened my skills in:

- Responsive layout design
- Theme management with CSS variables
- Bootstrap customization
- Code organization and maintainability
- Technical presentation in English
